JS函数的执行时机

Written by tangallison2 with ♥ on July 8, 2022 in
  • JS函数的执行时机不同，意味着不同的结果,我们举个例子。
  • setTimeout()方法设置一个定时器，该定时器在定时器到期后执行一个函数或指定的一段代码。我们可以把setTimeout当做尽快执行的意思，尽快执行的前提是把当前手头的事情先做完。
  • 例子一
let i =0
for(i=0;i<6;i++){
    setTimeout(()=>{
        console.log(i)
    },0)
}

  • 运行结果是6个6

  • 分析一下为什么结果是6个6，而不是0、1、2、3、、4、5？ setTimeout它要把for循环代码执行完再打印出i，for循环执行i=6时，才能跳出for循环，此时i已经执行了6次，i的值是6，等到setTimeout来执行console.log(i)时，得到的结果就是i=6打印6次。

  • 例子二 打印出0、1、2、3、4、5的方法

for(let i=0;i<6;i++){
    setTimeout(()=>{
        console.log(i)
    },0)
}

用for和let配合就可以。

  • 例子三 除了使用 for let 配合，还有什么其他方法可以打印出 0、1、2、3、4、5？
for(i=0;i<6;i++){
    console.log(i)
}

直接用for就可以
