Js对象的基本用法

Written by tangallison2 with ♥ on June 16, 2022 in

《JS 对象基本用法》

  1. 声明对象的两种语法
let obj = { 'name': 'tang', 'age': 18 }

let obj = new Object({'name': 'tang'})
  1. 如何删除对象的属性
delete obj.xxx
delete obj['xxx']
  • 验证对象是否存在属性’xxx’,若显示false则不存在
'xxx' in obj === false
  • 对象含有属性名，但是属性值为undefined的情况

‘xxx’ in obj && obj.xxx ===undefined

  • 注意obj.xxx===undefined不能判断’xxx’是否为obj的属性
  1. 如何查看对象的属性
obj.xxx
obj['xxx']

查看对象的所有属性名

Object.keys(obj)

查看对象的所有属性值

Object.values(obj)

查看对象的属性

console.dir(obj)

Object.entries(obj)
  1. 如何修改或增加对象的属性

已有属性则改，没有属性则增。

  • 修改或者增加对象的自身属性
obj.name = 'tang'
obj['name'] = 'tang'

或者

let key = 'name'
obj[key] = 'tang'

key是变量，obj[key]是变量求值，相当于obj[‘name’]

批量增加属性

Object.array(obj,{'name':'tang','age':26})
  • 修改或者增加对象的共有属性

第一种方法：可以了解但是不推荐使用

obj.__proto__.toString = 'xxx'

第二种方法：

Object.prototype.toString = 'xxx'
  • 修改或者增加对象的原型
let common = {'name':'tang'}

let obj = Object.create(common)

或者(只需要了解，不推荐使用含有__proto__代码)

obj.__proto__ = common
  1. ‘name’ in obj和obj.hasOwnProperty(‘name’) 的区别

‘name’ in obj 无法看出对象自身属性还是共有属性

obj.hasOwnProperty('name') === true

表示’name’是obj的自身属性
